Project Overview
Handwritten Digit Recognition using CNN

This project implements a Convolutional Neural Network (CNN) for recognizing handwritten digits from the MNIST dataset. The system achieves high accuracy through deep learning techniques.

Technology Stack
  • TensorFlow/Keras
  • OpenCV
  • NumPy
  • Flask
  • Bootstrap 5
  • Matplotlib
  • Python 3.10
  • HTML5/CSS3
Model Architecture

The CNN architecture includes:

  • Convolutional layers for feature extraction
  • Max pooling layers for dimensionality reduction
  • Dropout layers to prevent overfitting
  • Dense layers for classification
  • Softmax activation for probability distribution
Dataset

The model is trained on the MNIST dataset, which contains 60,000 training images and 10,000 test images of handwritten digits (0-9). Each image is 28x28 pixels in grayscale.
